# research/ — 풀 학습 설계 근거 (Q1–Q5)

> 프로젝트 전체 개요·관통 주제·핵심 설계 결정 요약은 루트 [`../../README.md`](../../README.md) 참고.
> 이 문서는 research 폴더의 **네비게이션 인덱스**(범례·문서맵·다음 단계)다.
> 권장 읽는 순서: 01 → 02 → 03 → 04 → 05 (Q4는 Q2·Q3을, Q5는 Q4를 참조).

스모크 다음 단계(full training) 설계 결정을 **데이터 + 선행연구 1차 출처**로 정당화하는 노트 모음. 각 문서는 "읽고 이해하는 것"이 목적이라 사람 말 + 비유로 썼고, 모든 주장에 출처 등급을 단다.

## 다음 단계 (열린 과제)
